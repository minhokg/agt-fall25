# agt-fall25
This repo contain all the project for Algorithmic Game Theory 2025 Fall

## Project Overview
- each project has its own folder (e.g. problem set 2 -> ps2)

## Prerequisites
- Python 3.11
- Git

## Setup
- run "pip install -r requirements.txt" in the terminal
- run "pre-commit install" in the terminal
- if you want to build the package locally (there should be no use case for this), run "python -m build"
- if you want to build the documentation locally (there should be no use case for this), run "cd docs" and then ".\make.bat html"
